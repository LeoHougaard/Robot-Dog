# Put The Simple Robot URDF Export Here

Copy the full Onshape URDF export into this folder.

Recommended final shape:

```text
sim/robots/simple_quad_v0/onshape_export/
  robot.urdf
  meshes/
    body.stl
    front_left_upper.stl
    ...
```

Do not copy only the `.urdf` file if it references meshes. Keep all referenced mesh files in the same relative layout that Onshape exported.
